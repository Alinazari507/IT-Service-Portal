# 🌐 IT-Service-Portal

Ein modernes, benutzerfreundliches Portal zur Verwaltung und Bestellung von IT-Dienstleistungen und Hardware.



## 🚀 Features
- **Service-Katalog:** Auswahl von Hardware (Lenovo, Apple) und Software (Microsoft 365, Adobe).
- **TCO-Logik:** Automatische Berechnung der Total Cost of Ownership.
- **Responsive Design:** Optimiert für Desktop und Mobilgeräte.
- **Warenkorb-System:** Einfaches Hinzufügen von Services mit "Jetzt anfordern".

## 🛠️ Technologien
- **Backend:** Python (Flask/Django)
- **Frontend:** HTML5, CSS3 (Modern UI), JavaScript
- **Version Control:** Git & GitHub
